git initi 			-> Cria um repo git na pasta atual
git status 			-> Mostra o status do trabalho
git add nome_arquivo 		-> Mostra o status do trabalho (de cada arquivo no repositorio)
git commit -m 			-> Ffaz commit do que estava para ser commitado
git log 			-> Mostra todo o histórico de commits
